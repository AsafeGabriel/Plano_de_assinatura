# Conecta Saúde

A React Native app built with Expo that connects patients to health professionals with a subscription model.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

3. Scan the QR code with the Expo Go app on your phone or run on an emulator.

## Features

- Connect patients to health professionals via chat and video calls
- Subscription-based model with multiple plans and durations

## Technologies

- React Native
- Expo
